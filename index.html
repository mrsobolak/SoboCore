SobolakCore RAWTools v1.1

SobolakCore RAWTools is a lightweight desktop utility for processing Sony .ARW RAW files. It extracts JPEG previews, embeds XMP metadata, and includes a cleanup tool for RAW and XMP files.

Features:
• JPEG preview extraction
• XMP embedding
• Batch processing
• RAW and XMP cleaner

SoboEncrypt v1

SoboEncrypt is a Python based file encryption utility by SoboCorp. It uses AES 256 encryption with a simple tabbed interface for encrypting and decrypting files. No installer, no telemetry, just a script you run when you need to lock something down.

Features:
• AES 256 GCM encryption
• Password based key derivation (PBKDF2)
• Encrypt and decrypt tabs
• Custom .sobo encrypted file format
• Pure Python script, no EXE required

SoboSuite v1

SoboSuite is a lightweight all in one desktop utility built in Python. It combines tasks, notes, a focus timer, file organization, and a quick launcher into one clean app. Session based by design. No file storage, no tracking, full control.

Modules:
• Tasks
• Notes with search
• Focus timer (Pomodoro)
• Auto organizer with preview mode
• Quick launcher for apps files and folders

My Story

My name is Sobo. My life started simple and quiet with small moments that showed me what normal life could have been.

Over time the world around me shifted. People fell into addiction. Calm turned into conflict. Stability disappeared.

I saw things no kid should ever witness. I learned to survive mentally and emotionally long before I should have.

Through everything I learned how to build. Electronics repair, GPU recovery, coding, automation, project creation. I rebuilt broken things because I grew up surrounded by things no one else knew how to fix.

Silence is not weakness. Silence is where vision takes shape.

People spend their lives trying to fit into existing systems. I would rather build new systems. I would rather invent my place than inherit one.

I stepped out of the cycle I was born into. I built my future from nothing. I am Sobo. I build. I fix. I create.
