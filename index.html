FAQ

Fast answers that build trust.

Does SoboAI need internet?
Only for the first model download. Then you can run offline.
Does it work without a GPU?
Yes. A GPU is recommended for speed, but CPU mode works fine too.
Can I export chats?
Yes. Export to TXT or Markdown.
Is my data stored online?
No. The point is keeping data local to your machine.
